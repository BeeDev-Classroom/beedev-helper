Installing required libraries

Before you start with data science in Python, you need to install some essential libraries.

Open your terminal or command prompt and type the following commands:

pip install numpy pandas matplotlib seaborn scikit-learn

importing required libraries

Once you have the libraries installed,

open your Python environment (IDE or Jupyter Notebook) and import them:

import numpy as np

import pandas as pd

import matplotlib.pyplot as plt

import seaborn as sns
